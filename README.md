# Slate Social

Slate Social is a social media platform engineered for climbing gyms. This repository contains the landing page website.

## Features

- Message boards for gym-wide discussions
- Groups & clubs management
- Competition hosting and leaderboards
- Gym news and announcements
- Polls and member feedback
- Partner finder for climbing buddies

Part of the Slate ecosystem.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
